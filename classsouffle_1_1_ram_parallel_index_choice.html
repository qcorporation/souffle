Detailed Description

Use an index to find a tuple in a relation such that a given condition holds in parallel.

For example:

QUERY
...
PARALLEL CHOICE A AS t1 ON INDEX t1.x=10 AND t1.y = 20
WHERE (t1.x, t1.y) NOT IN A
...
